# To_Do_App
# 📝 Advanced Todo List App  A modern and responsive Todo List application built using HTML, CSS, and JavaScript. The app stores tasks in Local Storage, allowing users to keep their tasks even after refreshing or closing the browser.  ## 🚀 Features  - Add new tasks - Edit existing tasks - Delete tasks - Mark tasks as completed - Search tasks instantly - Filter tasks:   - All Tasks   - Active Tasks   - Completed Tasks - Clear all completed tasks - Task counter - Local Storage support - Responsive design - Modern Glassmorphism UI  ## 📂 Project Structure  ```text todo-app/ │ ├── index.html ├── style.css ├── script.js └── README.md ```  ## 🛠 Technologies Used  - HTML5 - CSS3 - JavaScript (ES6) - Local Storage API  ## 💻 Installation  1. Clone the repository  ```bash git clone https://github.com/yourusername/todo-app.git ```  2. Open the project folder  ```bash cd todo-app ```  3. Open `index.html` in your browser  No additional dependencies or installations are required.  ## 📱 Usage  - Enter a task in the input field. - Click **Add** or press **Enter**. - Use filters to view specific tasks. - Click **Edit** to modify a task. - Click **Delete** to remove a task. - Mark tasks as completed using the checkbox. - Completed tasks are automatically saved in Local Storage.  ## 🎯 Future Improvements  - Dark / Light Mode - Drag & Drop Task Sorting - Due Dates - Priority Levels - Categories - PWA Support - Cloud Sync - User Authentication  ## 📸 Screenshots  Add screenshots here after deployment.  ## 🌐 Live Demo  Deploy on:  - Netlify - Vercel - GitHub Pages  ## 📄 License  This project is licensed under the MIT License.  ## 👨‍💻 Author  Developed by Your Name.
